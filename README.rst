sample-aspects plugin for `Tutor <https://docs.tutor.edly.io>`__
################################################################

sample-aspects plugin for Tutor


Installation
************

.. code-block:: bash

    pip install git+https://github.com/saraburns1/tutor-contrib-aspects-sample

Usage
*****

.. code-block:: bash

    tutor plugins enable sample-aspects
